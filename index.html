Help Guide

Hey there! Welcome to the "HELP" page. Here, you can see how each and every function of this website works.

  1. From the dropdown menu on the top right side, you can sort how your tasks are displayed. The buttons are pretty-much self explanatory.
  2. On the top right corner, there is an icon showing moon/sun. Clicking them will switch between light and dark mode. (This saves to database so you don't have to worry about changing it every time you visit the site)
  3. Below it are stats. These stats show total tasks/pending tasks/completed tasks/overdue.
  4. Under that, there are quick action buttons. You can either show all/active/completed tasks or even delete every completed task at once by clicking the "Clear Completed" button. (clear completed will never delete pending tasks, don't worry)
  5. Beneath it is an important category, the "Add New Task". This is mainly the heart of the website. Beside the text is a button either saying "Collapse" or "Expand". When it is pressed, the category will open or close. (It has an animation, so please don't spam the button or it might glitch). It has: Task Title, Due Date, Description, Difficulty, Importance, Checkpoints. For the checkpoints part, you MUST add at least one or else the system which automatically moves tasks from "To-do" to either "In Progress" or "Completed" won't work and you'd have to drag and drop it manually. In order to add new checkpoints, you have to create a new line. In pc/laptop, you can hit enter and it'll create new lines. Each line is a new checkpoint.
  6. Finally, the main purpose of the website, the tasks. This is where every single one of your tasks are displayed in the corresponding tabs. Please note that, tasks can be dragged and dropped from one tab to another. If you click on a task, it will expand and show details. If you click the edit button on the task, it will display your task in the "Add New Task" category. From there, you can edit anything you need and then hit save.

Do remember, every single necessary function saves to the database. So, you don't have to worry about your tasks being magically disappeared.

This was all, thank you so much for visiting the site.
